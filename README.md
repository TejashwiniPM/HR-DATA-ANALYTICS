📊 HR Data Analytics Dashboard (Power BI)


📌 Project Overview

This project focuses on analyzing Human Resources (HR) data using Power BI to uncover valuable insights about employee attrition, workforce distribution, and key HR metrics.
The interactive dashboard helps identify patterns and trends that can support data-driven decision-making in HR management, especially in improving employee retention and understanding workforce demographics.


🎯 Problem Statement

Employee attrition is one of the major challenges for organizations. This project aims to analyze HR data to understand:
What factors contribute to employee attrition
Which departments have higher turnover
Workforce demographics and distribution
Key HR performance indicators
The goal is to transform raw HR data into clear and actionable insights.

🛠 Tools & Technologies Used

Power BI – Data visualization and dashboard creation
Power BI – Data cleaning and preprocessing
Data Analysis – Identifying patterns and trends

📂 Dataset Description

The dataset contains employee-related information such as:
Employee ID
Age
Gender
Department
Job Role
Salary / Income
Years at Company
Attrition Status
This data was used to analyze workforce patterns and employee behavior.

📊 Dashboard Features

The Power BI dashboard provides visual insights including:
Overall Employee Attrition Rate
Attrition by Department
Attrition by Age Group
Employee distribution by Job Role
Gender diversity analysis
Average income by job role
Interactive filters allow users to explore the data dynamically.

🔍 Key Insights

Some insights discovered from the analysis include:
Certain departments experience higher attrition rates.
Employees with fewer years at the company are more likely to leave.
Workforce distribution varies significantly across job roles.
Salary and job role may influence employee retention.
